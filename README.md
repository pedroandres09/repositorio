# Estafa bizum
Es el “falso Bizum por error”, alguien envía dinero supuestamente por equivocación y pide que lo devuelvas; aunque se alerta de posibles pérdidas, no es posible que al devolverlo te clonen la cuenta, ya que Bizum requiere tus credenciales y no comparte datos bancarios con terceros. Para evitar estos fraudes, siempre conviene comprobar en la app bancaria que el ingreso realmente se ha recibido, desconfiar de devoluciones urgentes y no aceptar solicitudes de pago sin verificar, y en caso de duda contactar con el banco y las fuerzas de seguridad.
